Manipulación del DOM

DOM (Document Object Model) es una representación estructurada de un document. Cada elemento de la página, desde el título hasta los párrafos y las imágenes, se convierte en un objeto al que JavaScript puede acceder y modificar.


¿Por qué es importante la manipulación del DOM?

  • Permite crear páginas web que respondan a las acciones del usuario, como mostrar u ocultar elementos, cambiar el contenido o aplicar estilos.
  • Se pueden desarrollar interfaces de usuario interactivas, donde el usuario se sienta más involucrado.
  • Desarrollo de aplicaciones web de una sola página (SPA), donde todo el contenido se carga en una sola página y se actualiza dinámicamente.

¿Qué se puede hacer con el DOM?

  • Seleccionar elementos del HTML por su ID, clase, etiqueta o cualquier otra característica.
  • Modificar el texto, los atributos o el estilo de los elementos.
  • Crear elementos en el DOM.
  • Eliminar elementos del DOM.
  • Organizar o anidar elementos.
  • Escuchar eventos: Detectar acciones del usuario, como clics, cambios de valor en un formulario, etc., y ejecutar código en respuesta.

Métodos y propiedades comunes

  • getElementById: Obtiene un elemento por su ID.
  • querySelector: Selecciona el primer elemento que coincida con un selector CSS.
  • querySelectorAll: Selecciona todos los elementos que coincidan con un selector CSS.
  • createElement: Crea un nuevo elemento.
  • appendChild: Agrega un elemento como hijo de otro.
  • removeChild: Elimina un elemento hijo.
  • textContent: Obtiene o establece el contenido de texto de un elemento.
  • innerHTML: Obtiene o establece el contenido HTML de un elemento.
  • classList: Maneja las clases de un elemento.
